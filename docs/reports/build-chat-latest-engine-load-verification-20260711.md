## 검증 중 발견·수정한 결함

1. **OpenAI Structured Outputs 전체 차단**
   - `boardFocusIntent.categories`에 OpenAI가 허용하지 않는 `uniqueItems`, `maxItems`가 포함돼 LLM 요청이 HTTP 400으로 실패했다.
   - 지원되지 않는 키워드를 제거하고 서버 후처리의 중복 제거·최대 8개 제한은 유지했다.
2. **위치 표현 누락**
   - `구성도에서 CPU 찾아줘`가 위치 의도로 인식되지 않았다.
   - `찾아`, `짚어`를 명확한 공간 탐색 동사로 추가했다.
3. **MIN 예산 하한 위반**
   - `400만원 이상` 요청이 218만~395만원 조합을 반환했다.
   - MIN 전용 상향 사다리를 추가하고 snapshot·LLM·fallback 모든 경로에서 하한 미달 카드를 차단했다.
4. **수랭/케이스 수식 관계 오판**
   - `360 수랭 장착 케이스`를 쿨러 추천으로 분류했다.
   - `케이스/쿨러` 핵심 명사의 문장 내 순서를 이용해 실제 대상 명사를 판정하도록 보강했다.
5. **QA clarification 오탐**
   - 정상 원문 에코가 있어도 assistant 본문의 카테고리 단어만 검사했다.
   - clarification 허용 경로는 `originalMessage`도 카테고리 보존 근거로 검사한다.
